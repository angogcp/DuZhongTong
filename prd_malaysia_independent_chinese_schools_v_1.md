# 马来西亚独中综合情报平台 PRD v1.1

## 一、项目名称
**独中通｜马来西亚全国独中择校与升学情报平台**

---

## 二、项目背景

马来西亚全国独中资讯分散在董总官网、各学校官网、Facebook 专页、招生简章、家长社群与升学文章中。家长与学生在选校时，通常会遇到这些问题：

- 学校很多，但资料不集中
- 各校介绍格式不同，难以比较
- 学费、宿舍、报考时间、统考表现、升学去向不容易查全
- 不知道哪所学校适合自己孩子
- 对 UEC / 统考 / 毕业升学路径理解不足
- 外州家长尤其缺乏宿舍与校园生活的真实资讯

因此，需要建立一个**全国独中的统一资料平台**，让用户可以：

- 找学校
- 比学校
- 看报考
- 查统考与升学
- 了解宿舍与校园生活
- 获取持续更新的独中情报

---

## 三、产品定位

一个面向家长、学生、校方与升学顾问的：

**“全国独中资料库 + 择校工具 + 报考资讯中心 + 升学情报平台”**

---

## 四、目标用户

### 1. 家长
需求：
- 为孩子挑选适合的独中
- 比较学费、宿舍、学术与校风
- 了解报考流程与升学路径

### 2. 学生
需求：
- 了解学校环境、课程与社团
- 查询宿舍、统考、未来升学方向
- 找到适合自己的学校

### 3. 校方
需求：
- 提升招生曝光
- 展示学校特色与优势
- 维护官方资料的准确性

### 4. 升学顾问 / 教育内容创作者
需求：
- 使用标准化数据做分析
- 快速查询学校与升学资料

---

## 五、产品目标

### 业务目标
- 建立全国独中标准化数据库
- 成为家长查询独中资讯的入口网站
- 提升平台自然搜索流量
- 后期发展招生导流与学校合作

### 用户目标
- 3 分钟内找到符合条件的学校
- 5 分钟内完成 2–4 所学校比较
- 快速理解 UEC 与升学路径
- 降低择校焦虑与资讯不对称

- 英文名称
- 州属
- 城市
- 地址
- 地图位置
- 联络电话
- Email
- 官网
- Facebook / Instagram / YouTube
- 创校年份
- 学校简介
- 办学理念
- 校训 / 教育特色
- 学制
- 是否有初中 / 高中
- 课程与选科
- 语言环境
- 学费
- 杂费
- 奖助学金
- 是否有宿舍
- 宿舍详情
- 设施
- 社团活动
- 招生时间

- 统考成绩摘要
- 毕业升学去向

用户可勾选多所学校并横向比较。

#### 比较字段建议
- 州属 / 地点
- 学费
- 宿舍
- 创校年份
- 学校规模
- 学制
- 统考表现
- 升学率 / 升学方向
- 校园设施
- 招生条件
- 社团活动
- 是否适合外州生

### 模块三：报考资讯中心
整合各校招生、报名、入学考试、开放日等资料。

#### 内容结构
- 招生时间总表
- 开放日 / 说明会
- 报名入口
- 入学考试日期
- 截止日期
- 申请材料清单
- 奖学金申请
- 宿舍申请流程
- 新生常见问题

### 模块四：UEC / 统考情报中心
系统化介绍独中统考，让家长与学生容易理解。

#### 内容建议
- 什么是 UEC
- 初中统考 / 高中统考
- 科目说明
- 计分方式
- 统考流程
- UEC 与 SPM / STPM / A-Level / IGCSE 对比

- 可申请大学与国家
- 常见误解
- 历年趋势图

### 模块五：毕业升学数据库
展示各校毕业生升学路径与趋势。

#### 内容维度
- 升学率
- 本地 / 海外升学比例
- 热门升学国家
- 热门科系
- 奖学金去向
- 升学案例
- 校友访谈

### 模块六：宿舍与校园生活
针对外州家长与住宿生提供高价值情报。

#### 内容字段
- 是否有宿舍
- 男 / 女宿舍
- 房型
- 宿舍费用
- 膳食安排
- 晚自习制度
- 舍监 / 宿舍管理
- 安全机制
- 周末制度
- 洗衣 / 医疗 / 网络
- 住宿生常见问题
- 学生生活照片 / 访谈

### 模块七：内容专题

#### A. 独中百科
- 独中是什么
- 独中适合什么样的学生
- 独中与国中 / 国际学校比较
- 读独中的优缺点

#### B. 报考指南
- 小六毕业如何选独中
- 外州生如何找有宿舍的学校
- 预算有限怎么选独中

#### C. 升学专题
- UEC 去台湾
- UEC 去中国
- UEC 去新加坡
- UEC 去英国 / 澳洲 / 日本

#### D. 新闻与动态
- 招生新消息
- 政策更新
- 学校开放日
- 董总相关教育资讯

## 八、建议增加的高价值功能

### 1. 智能筛选找学校
用户可以按以下条件筛选：
- 州属
- 城市
- 学费范围
- 是否有宿舍
- 男女校 / 混校
- 偏学术 / 偏全人教育
- 外州生友好度
- 升学导向强弱

### 2. 学校推荐问答工具
用户填写几个问题后，系统推荐合适学校。

示例问题：
- 预算多少？
- 是否需要宿舍？
- 偏好哪个州？
- 更重视成绩还是活动？
- 想找纪律严格还是环境自由一点的学校？

输出：
- 推荐学校列表
- 推荐理由
- 可进一步比较

### 3. 学费计算器
不只显示学费，而是显示总成本。

#### 计算项目
- 报名费
- 注册费
- 学费
- 宿舍费
- 膳食费
- 校车费
- 杂费
- 一次性费用

输出：
- 每月估算
- 每年估算
- 第一年总成本

### 4. 招生日历
用日历方式展示：
- 开放日报名
- 说明会
- 入学考试
- 报名截止日
- 奖学金截止日

### 5. 地图找学校
全国地图按州属展示学校分布。

支持：
- 点击州属查看学校
- 是否有宿舍筛选
- 学费标签
- 热门学校标签

### 6. 家长 / 学生 / 校友故事
与其做粗暴打分，不如做内容型评价。

形式：
- 家长投稿
- 学生一天校园生活
- 校友升学分享
- 为什么我选择这所独中

### 7. 校方认领与资料更新
学校可以申请“认领学校页面”。

可更新内容：
- 招生资料
- 学费
- 宿舍
- 学校照片
- 统考成绩摘要
- 升学数据
- 活动讯息

## 十一、用户流程

### 流程一：家长找学校
1. 进入首页
2. 使用搜索或筛选条件
3. 浏览学校列表
4. 打开学校详情页
5. 收藏或加入比较
6. 前往比较页查看差异
7. 点击学校官网或报名入口

### 流程二：外州家长找有宿舍学校
1. 进入学校列表页
2. 勾选“有宿舍”
3. 选择州属 / 预算
4. 查看宿舍详情与费用
5. 对比多所学校
6. 联系学校或提交咨询

### 流程三：学生了解 UEC 升学路径
1. 进入 UEC 专题页
2. 查看 UEC 介绍与科目说明
3. 浏览升学国家页
4. 查看学校毕业去向
5. 形成选校和升学认知
